The `database` command is used for managing database services within Andasy. It allows you to provision and control supported databases as part of your application infrastructure.

Currently, the following database engines are supported:

- [PostgreSQL](/docs/services/databases/postgres) â€” Manage PostgreSQL as a service in your apps


> More database engines will be supported in future versions of Andasy.

## Usage

```bash
andasy database
```

You can also use any of the following aliases:
- db
- database

To view help for this command:

```bash
andasy database -h
```
